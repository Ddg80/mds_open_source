# Bienvenue dans notre projet Open Source!
Notre projet open source permet de créer des shorts cut d'url.
# Installations
```bash
npm i
```
## Dépendances installed
```Bash
 npm i dotenv express mongoose nanoid
 npm i -D nodemon
```
## Run application
```Bash
npm run start
```
